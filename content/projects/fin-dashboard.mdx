The Finance Dashboard is a dynamic web application that provides insights into financial data. Built using Next.js, it demonstrates modern web development techniques and an interactive user interface for displaying financial metrics.

## Features

- **Interactive Charts**: Visualize financial data through responsive charts.
- **Real-time Updates**: Dynamically update metrics and visuals.
- **User-Friendly Interface**: A clean and intuitive design for seamless navigation.
- **Deployed on Vercel**: Experience fast and reliable hosting.

## Technologies

- **Next.js**: Framework used to build the application.
- **React**: Frontend library for building the user interface.
- **Tailwind CSS**: Styling the application with modern and responsive design.
- **Chart.js**: Used for creating interactive charts.
- **Vercel**: Platform for deployment.

## Live Demo

Explore the live application: [Finance Dashboard](https://nextjs-finance-dashboard-sigma.vercel.app/).

## Getting Started

To get started with the Finance Dashboard project, follow these steps:

### Prerequisites

Ensure you have the following installed on your machine:

- Node.js (v18 or later)
- npm or pnpm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/shar3579/nextjs-finance_dashboard.git
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Start the development server:
   ```bash
   pnpm run dev
   ```

4. Open the application in your browser:
   ```
   http://localhost:3000
   ```
